## 🚀 快速开始

### 1. 启动服务
```bash
python -m uvicorn backend.app:app --host 0.0.0.0 --port 8000 --reload
```

### 2. 访问前端
打开浏览器：**http://localhost:8000**

### 3. 选择基础模型

在"模型训练"页面，您会看到：

**训练模式**下拉框：
- **从头训练** - 使用 YOLO11m 预训练模型（推荐）
- **增量学习** - 从已完成的训练继续

**基础模型**下拉框（选择"从头训练"时显示）：
- `yolo11m.pt [官方]` - 官方预训练模型
- `models/yolo11m.pt [本地]` - 您刚放入的模型
- 已完成的训练任务（如果有）

### 4. 配置参数
```
训练轮数：100
批次大小：8
图片尺寸：640
```

### 5. 开始训练
点击"开始训练"按钮
